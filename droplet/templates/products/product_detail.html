    Redis is the abbreviation of Remote Dictionary Server. The original developer of Redis is Salvatore Sanfilippo who was hired by VMware in March, 2010. Redis is a key-value data stored system which is based on memory. Redis is also an open source software and it usually is considered data structure server. Redis support lots of stored value types, including string, list, set, zset (sorted set) and hash. Besides, it provides Java，C/C++，C#，PHP，JavaScript，Perl，Object-C，Python，Ruby，Erlang and other programming languages. Moreover, compared to other NoSQL databases, it is easier to set up the performance level for Redis.
